Our Project

The project involves creating a website to display real-time CPU temperature on a dynamic graph. The website will be hosted on a server, making it accessible online for remote monitoring. This ensures continuous tracking of the CPU's thermal performance. We are also using the W3.CSS framework to style our web application and make it responsive.

Our Goal

The goal of this project is to develop a website that continuously monitors and displays the CPU temperature in real-time using a dynamic graph. This allows for efficient and immediate tracking of thermal performance, helping to prevent overheating issues. Once developed, the website will be deployed on a server, ensuring it is accessible online for remote monitoring and analysis.

Website founded est 2024.
